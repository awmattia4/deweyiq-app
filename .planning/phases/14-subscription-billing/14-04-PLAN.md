  <action>
    Add to existing `src/actions/billing.ts`:

    1. `createPortalSession()` — creates Stripe Customer Portal session
       - Returns { url } for redirect to Stripe's hosted portal
       - Portal handles payment method updates and invoice downloads

    2. `changePlan(tier: BillingTier, interval: BillingInterval)` — changes subscription tier/interval
       - Creates a new Stripe Checkout Session (or uses stripe.subscriptions.update for existing subs)
       - Stripe handles proration automatically
       - Returns { url } for Checkout redirect OR { success } for inline update

    3. `cancelSubscription()` — sets cancel_at_period_end on Stripe subscription
       - Does NOT immediately cancel — subscription stays active until period end
       - Returns { success, currentPeriodEnd }

    4. `reactivateSubscription()` — removes cancel_at_period_end
       - Only works if subscription is still active (period hasn't ended)
       - Returns { success }

    5. `getInvoices()` — fetches last 12 invoices from Stripe API
       - Returns array of { id, date, amount, status, pdfUrl, hostedUrl }
       - Calls stripe.invoices.list with customer ID and limit 12
  </action>
