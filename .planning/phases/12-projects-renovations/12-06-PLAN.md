2. Proposal number + version + date
3. Customer info (name, address)
4. Project info (type, description)
5. Scope description section
6. If show_line_item_detail=true: itemized line items table with category, description, qty, unit price, total grouped by category. If false: summary only (total per category, no line detail).
7. Good/Better/Best tier comparison table (if tiers exist): columns showing tier name, description, features, price. Matching the side-by-side style.
8. Add-ons section (list with prices)
9. Payment schedule table: milestone name, amount, retainage held, net amount
10. Terms & Conditions section
11. Warranty info section
12. Cancellation policy section
13. Signature block area (placeholder -- actual signature captured digitally on approval page)
14. "Powered by DeweyIQ" footer

Use the same hex color palette defined in quote-pdf.tsx. Same font sizes, padding, and layout conventions for brand consistency.
