Fetch all customers for the org using `withRls` with a pool count subquery (Pattern 7 from RESEARCH.md):
```typescript
const result = await withRls(token, (db) =>
  db
    .select({
      id: customers.id,
      full_name: customers.full_name,
      address: customers.address,
      phone: customers.phone,
      route_name: customers.route_name,
      status: customers.status,
      pool_count: sql<number>`(SELECT COUNT(*) FROM pools WHERE pools.customer_id = ${customers.id})`.as("pool_count"),
    })
    .from(customers)
    .orderBy(asc(customers.full_name))
)
```
