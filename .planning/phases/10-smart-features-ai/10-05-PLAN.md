    **Service frequency recommendation** (in both dialogs):
    - Based on pool type and volume:
      - Pool > 20,000 gallons: suggest "Weekly service recommended"

      - Spa: suggest "Weekly service recommended (spas need more frequent chemical attention)"
      - Pool with attached spa: suggest "Weekly for both"
    - Show as a subtle hint below the frequency field, not a hard constraint.

    **Equipment combo suggestions** (in add-pool-dialog.tsx when equipment section is visible):

    - Show as clickable "Quick add" chips that pre-fill equipment type when tapped.

    **Gate code reminder**:
    - If the address is filled but gate_code is empty, show a subtle prompt: "Does this property have a gate? Adding the code now saves techs time."

    Style all suggestions as `text-xs text-muted-foreground` with a lightbulb icon (Lightbulb from lucide-react), dismissible via an X button. Store dismissed suggestions in component state so they don't reappear during the same dialog session.
