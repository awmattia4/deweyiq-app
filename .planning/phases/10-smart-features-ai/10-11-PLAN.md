  <name>Task 1: Notification bell, panel, and server actions</name>
  <files>src/components/notifications/notification-bell.tsx, src/components/notifications/notification-panel.tsx, src/actions/user-notifications.ts, src/components/shell/app-header.tsx</files>
  <action>
    **Server actions** — Create `src/actions/user-notifications.ts`:
    - `getNotifications(limit?: number)`: Returns user's notifications ordered by created_at desc. Filters dismissed_at IS NULL. Groups into needs_action and informational arrays. Uses withRls.
    - `getUnreadCount()`: Returns count of notifications where read_at IS NULL and dismissed_at IS NULL. Uses withRls.
    - `markRead(notificationId: string)`: Sets read_at = now(). Uses withRls.
    - `markAllRead()`: Sets read_at = now() on all unread notifications for user. Uses withRls.
    - `dismissNotification(notificationId: string)`: Sets dismissed_at = now(). Uses withRls.

    **NotificationBell** — Create `src/components/notifications/notification-bell.tsx`:
    - Client component. Props: `{ userId: string; initialCount: number }`.
    - Renders Bell icon (lucide-react) with a red badge showing unread count (hidden when 0).
    - On mount, subscribe to Supabase Realtime postgres_changes on `user_notifications` table filtered by `recipient_id=eq.${userId}` for INSERT events (per research: only subscribe to INSERT, not UPDATE/DELETE due to REPLICA IDENTITY limitation).
    - On INSERT event, increment unread count.
    - On click, open the NotificationPanel.

    **NotificationPanel** — Create `src/components/notifications/notification-panel.tsx`:
    - Sheet (side="right") triggered by NotificationBell click.
    - Header: "Notifications" with "Mark all read" button.
    - Two sections: "Needs Action" (rendered first, bordered highlight) and "Informational" (below).
    - Each notification row: title, body (truncated), time ago, unread indicator (dot).
    - Clicking a notification: calls `markRead(id)`, navigates to `notification.link` if present.
    - Dismiss button (X) on each notification: calls `dismissNotification(id)`.
    - Empty state: "No notifications" in muted text.

    **App header integration** — Update `src/components/shell/app-header.tsx`:
    - Import and render `<NotificationBell>` in the header, positioned to the left of the user avatar/dropdown.
    - Pass userId and initial unread count from server props (add to the header component's props or fetch in the parent layout).
  </action>
