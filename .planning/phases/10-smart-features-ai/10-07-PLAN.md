      - `rain`: CloudRain icon + "Rain" label, blue-ish background.

    - StopCard renders `<WeatherBadge type={weather.type} label={weather.label} />` in the header area of the card, next to the stop index/name.
    - Since this is a daily-level forecast (not per-stop hourly), all stops show the same badge. The badge serves as a heads-up for the tech about today's conditions.
