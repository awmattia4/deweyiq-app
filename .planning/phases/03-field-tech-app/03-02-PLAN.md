  <behavior>
**CSI/LSI Calculator (lsi.ts):**

The CSI formula from Trouble Free Pool Wiki accounts for ionic strength, CYA correction, borate, salt, and temperature. It is more accurate than classic 5-factor LSI.

Expected behavior:
- `calculateCSI({ pH: 7.5, totalAlkalinity: 80, calciumHardness: 300, cya: 40, salt: 0, borate: 0, temperatureF: 80 })` → approximately -0.29 (balanced)
- `calculateCSI({ pH: 7.2, totalAlkalinity: 60, calciumHardness: 200, cya: 30, salt: 0, borate: 0, temperatureF: 70 })` → approximately -0.73 (corrosive)
- `calculateCSI({ pH: 7.8, totalAlkalinity: 120, calciumHardness: 400, cya: 50, salt: 3000, borate: 0, temperatureF: 85 })` → approximately +0.35 (scaling)
- Returns `null` when any required parameter (pH, totalAlkalinity, calciumHardness, temperatureF) is null
- Returns `null` when CarbAlk or CH <= 0
- Optional parameters (cya, salt, borate) default to 0 when null

`interpretCSI(csi)` returns:
- `{ status: "corrosive", label: "Corrosive", color: "red" }` when csi <= -0.6
- `{ status: "low", label: "Slightly Corrosive", color: "yellow" }` when -0.6 < csi <= -0.3
- `{ status: "balanced", label: "Balanced", color: "green" }` when -0.3 < csi <= +0.3
- `{ status: "high", label: "Slightly Scaling", color: "yellow" }` when +0.3 < csi <= +0.6
- `{ status: "scaling", label: "Scaling", color: "red" }` when csi > +0.6

**Target Ranges (targets.ts):**

Ideal chemistry ranges vary by sanitizer type. Export `getTargetRanges(sanitizerType)`:

| Parameter | Chlorine | Salt | Bromine |
|-----------|----------|------|---------|
| Free Chlorine | 2-4 ppm | 2-4 ppm | N/A |
| Bromine | N/A | N/A | 3-5 ppm |
| pH | 7.2-7.8 | 7.2-7.8 | 7.2-7.8 |
| Total Alkalinity | 80-120 | 80-120 | 80-120 |
| CYA | 30-50 | 60-80 | N/A |
| Calcium Hardness | 200-400 | 200-400 | 200-400 |
| TDS | 0-1500 | 2700-3400 (salt range) | 0-1500 |
| Phosphates | 0-200 | 0-200 | 0-200 |

`classifyReading(param, value, sanitizerType)` returns `{ status: "low" | "ok" | "high", color: "red" | "green" | "red" }` based on where value falls relative to target range.

**Product-Aware Dosing Engine (dosing.ts):**

Base dose rates per 10,000 gallons to change by 1 ppm (from CPO/industry standards):
- Sodium Hypochlorite 12%: 10.7 fl oz per 1 ppm per 10k gal (liquid chlorine)
- Calcium Hypochlorite 67%: 2.0 oz per 1 ppm per 10k gal (cal-hypo granular)
- Sodium Bicarbonate: 1.4 lbs per 10 ppm per 10k gal (raises alkalinity)
- Muriatic Acid 31%: ~26 fl oz per 10 ppm per 10k gal (lowers pH/alkalinity)
- Soda Ash: 6.3 oz per 0.1 pH per 10k gal (raises pH)
- CYA (Cyanuric Acid): 13 oz per 10 ppm per 10k gal

`calcDose(deltaPpm, volumeGallons, product)`:
- Scales linearly with volume (volumeGallons / 10000)
- Adjusts for product concentration: `referenceConcentration / product.concentrationPct`
- Returns `{ amount: number, unit: "floz" | "lbs" | "oz" }` rounded to 1 decimal place
- Example: `calcDose(2, 15000, { chemical: "sodiumHypochlorite_12pct", concentrationPct: 10 })` → higher amount because 10% is weaker than reference 12%

`generateDosingRecommendations(readings, pool, products)`:
- Takes current readings, pool info (volume, sanitizer type), and available products
- For each out-of-range parameter, calculates how many ppm to adjust and which product to use
- Returns array of `{ chemical, product, amount, unit, action, reason }` objects
- Example: pH is 7.9 (high), alkalinity is 80 (ok) → recommends muriatic acid to lower pH
