**2. Create Supabase Edge Function** `supabase/functions/send-service-report/index.ts`:

```typescript
// Deno Edge Function for sending service report emails via Resend
```

- Reads `RESEND_API_KEY` from `Deno.env.get()`
- Receives POST body: `{ visitId, customerEmail, customerName, reportHtml, fromName, fromEmail }`
- Sends via Resend API:
  - From: `{fromName} <{fromEmail}>` (or fallback "PoolCo <reports@poolco.app>")
  - To: customerEmail
  - Subject: "Service Report — {customerName} — {date}"
  - HTML body: reportHtml
- Returns 200 on success, 500 on failure
- Edge Function is idempotent — duplicate calls for the same visitId should not send duplicate emails
  - Check: if `email_sent_at` is already set on the visit, skip sending
  - After successful send: update `service_visits.email_sent_at` via direct Supabase client (service role in Edge Function)
- CORS headers for invocation from Next.js server action
